Autoclean timeout (s):
Automatically destroys all buffered messages after this many seconds of inactivity. Set to 0 to disable.
 The component postpones incoming messages. Messages accumulate in a buffer and can be released manually. - **Release First** — sends the oldest buffered message (FIFO) - **Release Last** — sends the newest buffered message (LIFO) - **Clean all** — destroys all buffered messages without sending ## Configuration - **Autoclean timeout (s)** — automatically destroys all buffered messages after this many seconds since the last message arrived. Set to `0` to disable. Default: `3600`.
