### Crontab [Advanced] Cron-based trigger component with support for multiple timers and OS command execution. **Features:** - Multiple cron timers with individual schedules - Execute OS commands on trigger (optional) - Output various data types: string, number, boolean, date, object, buffer, command output - Random data generation option - Optional trigger 10 seconds after initialization **Outputs:** - **Output**: Configured data or command stdout - **Error**: Command execution errors with stderr details
