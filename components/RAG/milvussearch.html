This component performs a similarity search against a Milvus-compatible vector store. Inputs - An embedding array (e.g. [0.12, -0.98, ...]). Outputs - The raw JSON response from the vector store, e.g.: { "code": 0, "cost": 0, "data": [ { "distance": 0.9, "id": 123, "language": "ua", "location": "Kyivska", "text": "Київська" }, { "distance": 0.80, "id": 456, "language": "ua", "location": "Lvivska", "text": "Львівська" } ], "topks": [ 2 ] } The settings allow configuring the instance, collection, vector field, metric type, result limit, and output fields to return.
