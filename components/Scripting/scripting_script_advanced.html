## Description This component executes a custom JavaScript code with support for multiple configurable inputs. Each input can have a custom name and optional path to read data from specific properties. ## Features - **Dynamic Inputs**: Add, rename, and configure multiple inputs - **Path Support**: Extract data from specific paths for each input - **Flexible Execution**: Supports both synchronous and asynchronous code - **Dual Outputs**: Send successful results to Output, errors to Error / False output
