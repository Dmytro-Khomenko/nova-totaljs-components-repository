# Jambonz Answer https://docs.jambonz.org/verbs/verbs/answer This component adds a Jambonz `answer` verb to the flow for answering incoming calls. ## Configuration - **Forward input data**: When enabled, preserves original input data alongside the jambonz_response array ## Behavior - Answers an incoming call by sending a `SIP 200 OK` - If `jambonz_response` array doesn't exist in input data, creates a new array with the answer verb - If `jambonz_response` array exists, appends the answer verb to the existing array - When "Forward input data" is disabled (default), only passes through `jambonz_response` without `forwarded_data` - When "Forward input data" is enabled, includes both `forwarded_data` and `jambonz_response` in output ## Input/Output Examples ### Standalone with Forward OFF (default): **Input:** `{ userId: "123", callId: "abc" }` **Output:** `{ jambonz_response: [{ verb: "answer" }] }` ### Standalone with Forward ON: **Input:** `{ userId: "123", callId: "abc" }` **Output:** ```json { "forwarded_data": { "userId": "123", "callId": "abc" }, "jambonz_response": [{ "verb": "answer" }] } ``` ### Sequential with Forward OFF: **Input:** `{ jambonz_response: [{ verb: "say", text: "Hello" }] }` **Output:** ```json { "jambonz_response": [ { "verb": "say", "text": "Hello" }, { "verb": "answer" } ] } ``` ### Sequential with Forward ON: **Input:** ```json { "forwarded_data": { "userId": "123" }, "jambonz_response": [{ "verb": "say", text: "Hello" }] } ``` **Output:** ```json { "forwarded_data": { "userId": "123" }, "jambonz_response": [ { "verb": "say", "text": "Hello" }, { "verb": "answer" } ] } ```
